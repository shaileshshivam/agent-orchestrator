What This PR Does

  • Adds v1 fork convergence primitives to the SCM contract.

  • Computes and exposes deterministic ahead/behind/diverged sync state.
  • Adds convergence suggestions (sync hint, upstreamable patch hint, drift alert).
  • Documents operator workflow for sync and divergence handling.

Architecture / Design Choices

  • Pure helper for state computation: computeForkSyncState.
  • Pure helper for suggestions: buildConvergenceSuggestions.
  • Single parser path for git count output to avoid duplicate parsing logic.
  • No implicit conflict merge in v1; diverged or ff-failure returns blocked.
  • Sync action remains explicit and explainable via returned state + suggestions.
